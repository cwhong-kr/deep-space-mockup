# Claude용 모바일 앱 목업 생성 프롬프트

아래 조건으로 **모바일 앱 목업 HTML**을 만들어줘.

## 프로젝트
- 서비스명: 전기차 워크 라운지
- 목적: 전기차 충전 시간 동안 사용할 수 있는 워크 라운지 예약 서비스
- 타겟 사용자: 충전 중 업무/휴식 공간이 필요한 직장인, 프리랜서, 장거리 운전자

## 산출물
- `index.html`
- `styles.css`
- `script.js`

## 구현 조건
1. 순수 HTML/CSS/JS만 사용
2. 외부 프레임워크 없이 동작
3. 모바일 퍼스트
4. iPhone 화면 비율에 맞는 목업 스타일
5. 전체 UI는 현대적이고 미니멀한 SaaS / mobility / premium lounge 톤
6. 더미 데이터 포함
7. 접근성을 고려한 시맨틱 태그 사용
8. 코드에 주석 포함

## 디자인 가이드
- 키워드: clean, premium, futuristic, calm, productivity
- 메인 컬러: 블루 계열 + 화이트 + 약한 민트 포인트
- 카드 UI 중심
- glassmorphism 또는 soft gradient 일부 사용 가능
- 텍스트 대비 충분히 확보
- 실제 앱처럼 status bar / bottom nav 포함

## 꼭 들어갈 화면 요소
- 상단 히어로 영역
- 빠른 액션 버튼
- 현재 예약 정보 카드
- 가까운 라운지 정보 카드
- 추천 공간 리스트
- 하단 내비게이션

## 코드 품질 요구사항
- CSS 변수 사용
- 컴포넌트처럼 재사용 가능한 클래스 네이밍
- 화면 확장 가능하게 구조화
- 나중에 React로 옮기기 쉬운 구조

## 출력 방식
- 파일별로 구분해서 출력
- 바로 복붙 가능한 완성 코드 제공
- 설명은 최소화하고 코드 중심으로 제공
