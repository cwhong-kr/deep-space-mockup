# Mobile Mockup Starter

전기차 워크 라운지 모바일 앱 목업을 빠르게 시작하기 위한 기본 세트입니다.

## 파일 구성
- `index.html` : 메인 목업 화면
- `styles.css` : 모바일 앱 스타일
- `script.js` : 간단한 인터랙션
- `claude_prompt.md` : Claude에 바로 넣을 프롬프트

## 사용 방법
1. `index.html`을 브라우저에서 열어 기본 목업 확인
2. `claude_prompt.md` 내용을 Claude에 입력
3. Claude가 생성한 코드로 현재 파일을 확장/교체
4. 필요한 경우 화면을 추가해서 `home`, `reservation`, `charging`, `profile` 등으로 확장
