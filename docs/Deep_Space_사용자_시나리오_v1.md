# Deep Space (DS) 사용자 이용 시나리오

> 버전: v0.1 초안 · 2026.04.22
> 목적: 사용자 관점 전체 플로우 및 앱 내 문구·Push 알림 문구 초안 정리

---

## 시나리오 전제

- 사용자: 전기차 보유, 코엑스 인근 미팅 예정인 직장인
- 상황: 미팅 2시간 전, 차량 배터리 42%, 미팅 전후로 업무 공간이 필요
- 서비스: Deep Space (DS) — 전기차 충전 + 업무 라운지 복합 서비스

## STEP 01. 소셜 로그인

### 화면 구성

- 앱 첫 실행 시 브랜드 스플래시 → 로그인 화면
- 소셜 로그인 버튼: 카카오 / 구글 / Apple ID
- 하단: 이메일·휴대폰 로그인 / 회원가입 링크

### 최초 가입 시 추가 입력

- 차량번호 (필수)
- 차종 선택 (전기차 모델 — 충전 예상 시간 계산에 필요)
- 푸시 알림 수신 동의

### 앱 내 문구 (안)

| 위치 | 문구 |
|------|------|
| 스플래시 | **DEEP — Time, Designed.** |
| 로그인 화면 타이틀 | 시간을 디자인하는 첫 걸음 |
| 로그인 화면 부제 | 3초 만에 시작하세요 |
| 차량등록 안내 | 차량번호를 등록하면 입차 체크인이 자동으로 진행됩니다 |

## STEP 04. 길 안내

### 화면 구성

- 예약 상세 화면 하단에 "길 안내" 버튼
- 네비게이션 앱 선택 바텀시트:
  - 티맵
  - 카카오내비
  - 네이버 지도
  - 차량 내비게이션 (차종별 분기)
- 선택 시 해당 앱으로 딥링크 전환 (목적지: 주차장 입구 좌표)

### 도착 시 충전 잔량 제공

- 현재 배터리 잔량 + 목적지까지 예상 소모량 → 도착 예상 잔량 표시

## 시나리오 전체 흐름 요약

```
소셜 로그인 → 지점 검색 → 지점 상세 확인 → 예약 & 결제
    → 길 안내 (네비 연동) → 지점 접근 (지오펜스 Push)
    → Push 클릭 → 예약 정보 화면 → DS 체크인 (QR/출입번호)
    → 이용 중 (충전 + 업무) → 종료 10분 전 Push
    → 퇴실 & 정산 → 출차 (지오펜스 Push)
```
